# Design Pattern[](http://www.zitiweb.com/temp/3544c28ff7a6a86bc00d2c59666ee6f5.png)

学习设计模式，共23种设计模式，学习过程中使用Java语言实现

设计原则：
 - 单一职责原则
 - 接口隔离原则
 - 依赖倒转原则
 - 里氏替换原则
 - 开闭原则 ocp
 - 迪米特法则
 - 合成服用原则
 
设计模式目的：
 - 重用性（相同功能的代码，不用多次编写）
 - 可读性（编程的规范性，便于代码的阅读和理解）
 - 拓展性（方便新功能的增加）
 - 可靠性（新增功能对旧功能不造城影响）
 - 高内聚、低耦合
